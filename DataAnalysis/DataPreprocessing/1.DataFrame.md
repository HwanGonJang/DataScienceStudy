## 결측치(NaN) 다루기

NaN은 그냥 사용하면 안되고 numpy 라이브러리를 import하고 사용해야 한다.

```python
import numpy as np
a = {'company' : ['abc', '회사', 123], '직원수' : [400, 10, 6], '위치' : ['Seoul', np.NaN, 'Busan']}
df4 = pd.DataFrame(a)

df4
```
