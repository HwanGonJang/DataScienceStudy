### deepcopy

오른쪽 사진처럼 복사 문제를 해결하기 위해 deep copy를 이용한다.

```
import copy
```



deepcopy로 복사한 후 df를 변경.

```python
df = pd.DataFrame({'a': [1, 2, 3], 'b' : [4, 5, 6], 'c' : [7, 8, 9]})
df2 = copy.deepcopy(df)

df.columns = ['d', 'e', 'f']
df2
```
